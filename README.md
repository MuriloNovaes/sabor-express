O "Sabor Express" é um programa de linha de comando que permite ao usuário cadastrar, listar e alternar o estado de atividade de restaurantes. Ele oferece uma interface simples para interagir com um banco de dados de restaurantes armazenado em uma lista de dicionários.

Funcionalidades
Cadastrar Restaurante: Permite ao usuário adicionar um novo restaurante ao sistema, especificando seu nome e categoria.
Listar Restaurantes: Exibe todos os restaurantes cadastrados, mostrando seu nome, categoria e estado (ativo ou inativo).
Alternar Estado do Restaurante: Permite ao usuário ativar ou desativar um restaurante existente.
Finalizar o Programa: Encerra a execução do programa.
Tecnologias Utilizadas
Linguagem: Python
Bibliotecas: Nenhuma biblioteca externa é utilizada além das padrão do Python.
Como Instalar
Certifique-se de ter o Python instalado em seu sistema.
Salve o código em um arquivo com a extensão .py, por exemplo, sabor_express.py.
Execute o programa a partir da linha de comando com python sabor_express.py.
Como Usar
Menu Principal: Ao iniciar o programa, o usuário é apresentado a um menu com opções para cadastrar, listar, alternar o estado de restaurantes ou sair.
Cadastro: Selecione a opção 1 para cadastrar um novo restaurante.
Listagem: Selecione a opção 2 para listar todos os restaurantes.
Alternar Estado: Selecione a opção 3 para ativar ou desativar um restaurante.
Sair: Selecione a opção 4 para finalizar o programa.
